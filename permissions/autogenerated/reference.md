## Default Permission

Default permissions for the drag-and-drop plugin

#### This default permission set includes the following:

- `allow-start-drag`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`drag-and-drop-wayland:allow-start-drag`

</td>
<td>

Enables the start_drag command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`drag-and-drop-wayland:deny-start-drag`

</td>
<td>

Denies the start_drag command without any pre-configured scope.

</td>
</tr>
</table>
